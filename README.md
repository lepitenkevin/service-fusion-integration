# Service Fusion API Test Integration (PHP)

This repository contains sample PHP files used for testing API integration with **Service Fusion**, including basic PHP execution and WordPress-based form testing.

## Files Included

- **index.php**  
  A basic PHP test file used to verify PHP setup and initial API connectivity testing.

- **wordpress.php**  
  A test file for integrating **Service Fusion API** with WordPress, using the **Contact Form 7** plugin to simulate form-based API requests.

## Purpose

This project is intended for:
- Testing Service Fusion API integration
- Validating API requests and responses
- Experimenting with PHP-based API calls
- Testing WordPress Contact Form 7 form submissions connected to Service Fusion
- Development, QA, and educational use

## Requirements

- PHP-enabled server (local or live)
- Service Fusion API credentials
- WordPress installation (for `wordpress.php`)
- Contact Form 7 plugin installed and activated

## Notes

- This repository is for testing and development purposes only.
- API credentials should not be committed to the repository.
- Not intended for production use.
